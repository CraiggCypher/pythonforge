# 🐍 Python Lab 1 Project

## Project Title: Student Information System

### Objective
Create a simple Python program that collects and displays a student's information using the concepts learned in Lab 1.

### Concepts Covered
✅ Python Syntax  
✅ Variables  
✅ Input and Output  
✅ String Slicing  
✅ Operators  
✅ String Methods  

---

## Project Requirements

Write a Python program that:

1. Asks the user to enter:
   - Full Name
   - Age
   - School
   - Favorite Subject

2. Displays the information in a neat format.

3. Converts the student's name to uppercase.

4. Displays the first 3 characters of the student's name.

5. Displays the last 3 characters of the student's name.

6. Calculates the student's age after 5 years.

7. Displays the total number of characters in the student's full name.

---
