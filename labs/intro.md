## 📚 Lessons

### 1. What is Programming?

Programming is the process of giving a computer a set of instructions to solve a problem
or perform a task. These instructions are written in a language the computer can understand,
called a **programming language**.

---

### 2. Why Python?

Python was created by **Guido van Rossum** and first released in **1991**. It is now one
of the most widely used programming languages in the world.

Python is used for:

- Web development (server-side)
- Software development
- Mathematics and data science
- System scripting and automation

**What can Python do?**

- Build web applications on a server
- Create workflows alongside other software
- Connect to databases, read and modify files
- Handle big data and perform complex mathematics
- Support rapid prototyping and production-ready software

**Why choose Python?**

- Works on all platforms — Windows, Mac, Linux, Raspberry Pi
- Simple syntax that reads close to plain English
- Write more with fewer lines of code than most other languages
- Runs on an interpreter — code executes as soon as it is written
- Supports procedural, object-oriented, and functional programming styles

---

### 3. Python Applications in Industry

Python is used across many industries including finance, healthcare, data science,
artificial intelligence, web development, and automation. Companies like Google,
Netflix, NASA, and Instagram rely on Python in their systems.
